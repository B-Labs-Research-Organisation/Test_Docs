# Nudl Installation Guide

Welcome to the Nudl setup guide.

This document explains how to install the local AI tooling required to run Nudl development workflows.

---

## Requirements

Before starting, ensure the following applications are installed:

- Docker Desktop
- Node.js LTS
- Flowise
- Ollama

Recommended system:
- Windows 10/11
- 16 GB RAM minimum

---

## Step 1 — Install Ollama

Download and install Ollama from the official website.

After installation, open PowerShell and run:
